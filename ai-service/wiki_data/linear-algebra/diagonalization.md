# 对角化

## 一、定义
对角化（Diagonalization）是线性代数中一种将方阵转化为对角矩阵的变换过程。对于一个n×n方阵A，若存在可逆矩阵P和对角矩阵D，使得：

**P⁻¹AP = D**

则称A可对角化，其中D的对角线元素为A的特征值，P的列向量为对应的特征向量。核心数学表达式为：

**A = PDP⁻¹**

对角化的本质是寻找一个基变换，使得线性变换在该基下的表示最为简洁——仅有对角线上的非零元素。

## 二、核心原理
1. **特征值与特征向量的对应关系**：对角化的基础是特征方程Av = λv。每个特征值λ对应一个特征向量v，对角矩阵D中的元素正是这些特征值。

2. **线性无关性要求**：矩阵可对角化的充要条件是存在n个线性无关的特征向量。这意味着几何重数（特征空间的维数）必须等于代数重数（特征值的重数）。

3. **基变换的几何解释**：矩阵P定义了从标准基到特征向量基的变换。在新基下，线性变换简化为各坐标轴方向的伸缩，无旋转或剪切分量。

4. **谱分解**：对于对称矩阵，对角化可进一步简化为正交对角化（P为正交矩阵），此时A = QΛQᵀ，称为谱分解。

5. **幂运算简化**：对角化的核心应用在于简化矩阵幂运算：Aᵏ = PDᵏP⁻¹，其中Dᵏ仅需对对角线元素分别求幂。

## 三、过程/示例
考虑矩阵 A = [[2, 1], [1, 2]]，演示对角化过程：

**步骤1：求特征值**
解特征方程 det(A - λI) = 0：
|2-λ, 1; 1, 2-λ| = (2-λ)² - 1 = 0
得 λ₁ = 3, λ₂ = 1

**步骤2：求特征向量**
λ₁ = 3：解 (A - 3I)v = 0
[[-1, 1], [1, -1]]v = 0 → v₁ = [1, 1]ᵀ

λ₂ = 1：解 (A - I)v = 0
[[1, 1], [1, 1]]v = 0 → v₂ = [1, -1]ᵀ

**步骤3：构造P和D**
P = [v₁ v₂] = [[1, 1], [1, -1]]
D = diag(3, 1) = [[3, 0], [0, 1]]

**步骤4：验证**
计算 P⁻¹ = (1/2)[[1, 1], [1, -1]]
P⁻¹AP = (1/2)[[1, 1], [1, -1]] × [[2, 1], [1, 2]] × [[1, 1], [1, -1]]
= [[3, 0], [0, 1]] = D ✓

## 四、复杂度/性质分析
1. **时间复杂度**：对角化的计算主要分为两步——求特征值和特征向量。一般n×n矩阵的完整对角化需要O(n³)次浮点运算（使用QR算法等迭代方法）。

2. **空间复杂度**：存储原始矩阵A需要O(n²)空间，加上特征向量矩阵P和对角矩阵D，总空间复杂度为O(n²)。

3. **数值稳定性**：当矩阵存在接近相等的特征值或病态特征向量时，对角化过程可能数值不稳定。对称矩阵的对角化通常更稳定。

4. **收敛性**：QR算法等迭代方法通常具有二次收敛性，对于对称矩阵可达三次收敛。但理论上，并非所有矩阵都可对角化（如Jordan块矩阵）。

5. **精度问题**：实际计算中，由于浮点误差，得到的D可能不是严格对角矩阵，而是接近对角的Hessenberg矩阵。

## 五、常见误区
1. **混淆可对角化与特征值存在性**：所有方阵在复数域都有特征值，但并非所有方阵都可对角化。例如Jordan块[[λ, 1], [0, λ]]有特征值λ但不可对角化。

2. **误认为不同特征值对应特征向量必正交**：仅对称矩阵的不同特征值对应的特征向量正交。一般矩阵的特征向量仅线性无关，不一定正交。

3. **忽略特征向量的归一化**：构造P时，特征向量可以任意缩放，但不同教材的归一化方式不同，导致P不唯一。计算P⁻¹AP时需保持一致性。

4. **混淆对角化与三角化**：Schur分解将矩阵化为上三角矩阵（酉三角化），而对角化要求更严格。所有方阵都可酉三角化，但仅部分可对角化。

5. **误用对角化于非方阵**：对角化仅定义于方阵。对于m×n矩阵，应使用奇异值分解（SVD）作为更一般的对角化形式。
