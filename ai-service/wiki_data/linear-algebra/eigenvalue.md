# 特征值

## 一、定义

特征值（Eigenvalue）是线性代数中描述线性变换固有特性的核心概念。对于一个给定的方阵 **A** ∈ ℂⁿˣⁿ，若存在非零向量 **v** ∈ ℂⁿ 和标量 λ ∈ ℂ，使得以下方程成立：

**A v** = λ **v**

则称 λ 为矩阵 **A** 的一个特征值，**v** 为对应于 λ 的特征向量。该方程称为特征方程。从几何角度看，特征向量在矩阵 **A** 的线性变换下仅发生伸缩变换，伸缩的倍数即为特征值。

特征值的求解通常通过求解特征多项式完成：det(**A** − λ**I**) = 0，其中 **I** 为单位矩阵，det 表示行列式。

## 二、核心原理

1. **几何意义**：特征向量定义了线性变换下方向不变的“主轴”。当矩阵作用于特征向量时，结果向量与原向量共线，仅长度按特征值缩放。正特征值表示同向伸缩，负特征值表示反向伸缩。

2. **代数多重性与几何多重性**：特征值 λ 的代数重数是其特征多项式中 (λ₀ − λ) 因子的幂次；几何重数是对应特征空间的维数（即 (A − λI) 的零空间维数）。几何重数 ≤ 代数重数，当两者相等时矩阵可对角化。

3. **谱分解**：若 n×n 矩阵 **A** 有 n 个线性无关的特征向量，则可对角化为 **A** = **PDP⁻¹**，其中 **D** 为特征值构成的对角矩阵，**P** 的列向量为对应特征向量。对于实对称矩阵，**P** 可取为正交矩阵。

4. **迹与行列式**：特征值之和等于矩阵的迹（主对角线元素之和），特征值之积等于矩阵的行列式。这一性质可用于快速验证特征值计算的正确性。

5. **相似变换不变性**：特征值在相似变换下保持不变。若 **B** = **P⁻¹AP**，则 **A** 与 **B** 具有完全相同的特征值集合。

## 三、过程/示例

考虑矩阵 **A** = [[2, 1], [1, 2]]，求解其特征值和特征向量。

**步骤1：建立特征方程**
det(**A** − λ**I**) = det([[2−λ, 1], [1, 2−λ]]) = 0

**步骤2：展开行列式**
(2−λ)(2−λ) − 1×1 = λ² − 4λ + 3 = 0

**步骤3：求解特征值**
λ₁ = 3，λ₂ = 1

**步骤4：求解特征向量**
对于 λ₁ = 3：求解 (A − 3I)**v** = 0
[[−1, 1], [1, −1]] **v** = 0 → v₁ = v₂
取 **v₁** = [1, 1]ᵀ（归一化后为 [1/√2, 1/√2]ᵀ）

对于 λ₂ = 1：求解 (A − I)**v** = 0
[[1, 1], [1, 1]] **v** = 0 → v₁ = −v₂
取 **v₂** = [1, −1]ᵀ（归一化后为 [1/√2, −1/√2]ᵀ）

**验证**：A **v₁** = [3, 3]ᵀ = 3 **v₁** ✓；A **v₂** = [1, −1]ᵀ = 1 **v₂** ✓

## 四、复杂度/性质分析

**时间复杂度**：
- 直接求解特征多项式（计算行列式）：O(n³) 至 O(n!)（取决于行列式算法）
- QR 算法（数值计算标准方法）：O(n³) 每次迭代，通常需要 O(n) 次迭代
- 幂法（求最大特征值）：O(n²) 每次迭代，收敛速度取决于特征值比值 |λ₂/λ₁|

**空间复杂度**：
- 存储原始矩阵：O(n²)
- 存储特征向量矩阵：O(n²)
- 迭代法通常需要 O(n) 额外空间

**收敛性**：
- 幂法线性收敛，收敛因子为 |λ₂/λ₁|
- QR 算法具有二次收敛性，对于对称矩阵可达三次收敛
- 对于病态矩阵（特征值接近），收敛速度显著下降

**数值稳定性**：
- 特征值问题通常是良态的，条件数 κ ≤ cond(V)（特征向量矩阵的条件数）
- 对于非正规矩阵，特征值可能对扰动极其敏感
- 实对称矩阵的特征值问题总是良态的

## 五、常见误区

1. **特征值与奇异值混淆**：特征值仅定义于方阵，而奇异值定义于任意矩阵。对于正定对称矩阵，奇异值等于特征值的绝对值；一般情况下两者无直接相等关系。

2. **零特征值的误解**：零特征值意味着矩阵奇异（不可逆），但反之不成立——奇异矩阵可以有非零特征值（如 Jordan 块中特征值为 0 但代数重数大于几何重数的情况）。

3. **特征向量的唯一性**：特征向量不是唯一的——任何非零标量倍仍然是特征向量。通常约定归一化（如单位长度），但方向（正负）仍不唯一。

4. **实矩阵必有实特征值**：实矩阵的特征多项式是实系数多项式，但根可能为复数。例如旋转矩阵 [[0, −1], [1, 0]] 的特征值为 ±i。实对称矩阵保证所有特征值为实数。

5. **对角化与可逆性混淆**：可对角化（存在 n 个线性无关的特征向量）与可逆性（所有特征值非零）是两个独立概念。例如，零矩阵可对角化但不可逆；Jordan 块 [[1, 1], [0, 1]] 可逆但不可对角化。
