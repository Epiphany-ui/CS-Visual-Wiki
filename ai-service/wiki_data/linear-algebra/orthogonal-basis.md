# 正交基

## 一、定义
正交基（Orthogonal Basis）是内积空间中一组特殊的基向量集合，其中任意两个不同的基向量彼此正交（内积为零）。若进一步每个基向量都是单位向量（范数为1），则称为标准正交基（Orthonormal Basis）。设向量空间 \(V\) 上的内积为 \(\langle \cdot, \cdot \rangle\)，一组基 \(\{v_1, v_2, \dots, v_n\}\) 是正交基当且仅当：
\[
\langle v_i, v_j \rangle = 0 \quad (i \neq j)
\]
若同时满足 \(\|v_i\| = 1\) 对所有 \(i\) 成立，则为标准正交基。在 \(\mathbb{R}^n\) 中，标准正交基的典型例子是标准单位向量组 \(\{\mathbf{e}_1, \mathbf{e}_2, \dots, \mathbf{e}_n\}\)。

## 二、核心原理
1. **内积为零的几何意义**：正交意味着向量在几何上相互垂直，在函数空间中则代表线性无关且无“重叠分量”。这一性质使得正交基在坐标表示时，每个坐标分量独立于其他分量。

2. **坐标计算的简化**：对于标准正交基 \(\{u_1, u_2, \dots, u_n\}\)，任意向量 \(x\) 的坐标可通过内积直接得到：\(x = \sum_{i=1}^n \langle x, u_i \rangle u_i\)。这避免了求解线性方程组，将坐标计算简化为 \(n\) 个独立的内积运算。

3. **Gram-Schmidt正交化**：从任意一组线性无关的向量出发，通过逐步减去已处理向量方向上的投影，可构造出正交基。核心公式为：
   \[
   v_k = a_k - \sum_{i=1}^{k-1} \frac{\langle a_k, v_i \rangle}{\langle v_i, v_i \rangle} v_i
   \]
   其中 \(a_k\) 是原始向量，\(v_i\) 是已得到的正交向量。

4. **保持范数与距离**：在标准正交基下，向量的范数等于其坐标向量的欧几里得范数（Parseval恒等式），即 \(\|x\|^2 = \sum_{i=1}^n \langle x, u_i \rangle^2\)。这一性质在信号处理和数值计算中至关重要。

## 三、过程/示例
考虑 \(\mathbb{R}^3\) 中的三个向量：\(a_1 = (1,1,0), a_2 = (1,0,1), a_3 = (0,1,1)\)。使用Gram-Schmidt过程构造正交基：

**步骤1**：取 \(v_1 = a_1 = (1,1,0)\)，计算其范数平方 \(\|v_1\|^2 = 2\)。

**步骤2**：计算 \(a_2\) 在 \(v_1\) 上的投影系数：
\[
\frac{\langle a_2, v_1 \rangle}{\|v_1\|^2} = \frac{1\cdot1 + 0\cdot1 + 1\cdot0}{2} = \frac{1}{2}
\]
得到 \(v_2 = a_2 - \frac{1}{2}v_1 = (1,0,1) - (0.5,0.5,0) = (0.5, -0.5, 1)\)。

**步骤3**：计算 \(a_3\) 在 \(v_1\) 和 \(v_2\) 上的投影：
\[
\frac{\langle a_3, v_1 \rangle}{\|v_1\|^2} = \frac{0\cdot1 + 1\cdot1 + 1\cdot0}{2} = 0.5, \quad \frac{\langle a_3, v_2 \rangle}{\|v_2\|^2} = \frac{0\cdot0.5 + 1\cdot(-0.5) + 1\cdot1}{1.5} = \frac{0.5}{1.5} = \frac{1}{3}
\]
其中 \(\|v_2\|^2 = 0.25 + 0.25 + 1 = 1.5\)。于是：
\[
v_3 = a_3 - 0.5v_1 - \frac{1}{3}v_2 = (0,1,1) - (0.5,0.5,0) - (\frac{1}{6}, -\frac{1}{6}, \frac{1}{3}) = (-\frac{2}{3}, \frac{2}{3}, \frac{2}{3})
\]
验证正交性：\(\langle v_1, v_2 \rangle = 0.5 - 0.5 + 0 = 0\)，\(\langle v_1, v_3 \rangle = -\frac{2}{3} + \frac{2}{3} + 0 = 0\)，\(\langle v_2, v_3 \rangle = -\frac{1}{3} - \frac{1}{3} + \frac{2}{3} = 0\)。得到正交基 \(\{v_1, v_2, v_3\}\)，进一步单位化可得标准正交基。

## 四、复杂度/性质分析
- **时间复杂度**：Gram-Schmidt正交化对 \(n\) 个 \(m\) 维向量，最坏和平均情况均为 \(O(n^2 m)\)，因为每个新向量需与之前所有向量计算内积。若 \(m = n\)，则为 \(O(n^3)\)。改进的修正Gram-Schmidt算法保持相同复杂度但数值稳定性更好。
- **空间复杂度**：若原地修改，额外空间为 \(O(1)\)；若保留原始向量，则为 \(O(nm)\)。
- **数值稳定性**：经典Gram-Schmidt对舍入误差敏感，可能导致正交性丧失；修正Gram-Schmidt通过逐步更新投影系数提高稳定性，但仍有累积误差。Householder变换和Givens旋转在数值上更稳定。
- **收敛性**：在精确算术下，Gram-Schmidt保证在有限步内完成正交化；在浮点运算中，正交性误差随向量数量增长，可通过重正交化策略控制。
- **唯一性**：正交基不唯一（可乘以任意非零标量），但标准正交基在相差一个正交变换的意义下唯一。

## 五、常见误区
1. **混淆正交与线性无关**：正交向量组必然线性无关，但线性无关组不一定正交。初学者常误以为任意线性无关组都是正交基，实际上需要额外正交化步骤。

2. **忽略内积定义**：正交性依赖于内积的定义。在 \(\mathbb{R}^n\) 中默认使用点积，但在函数空间（如 \(L^2[a,b]\)）中，内积定义为积分形式，正交性判断需使用对应内积。

3. **Gram-Schmidt顺序依赖**：正交化结果依赖于向量输入顺序。改变顺序会得到不同的正交基，但张成的子空间相同。初学者常误以为结果唯一。

4. **误用标准正交基的坐标公式**：坐标公式 \(x_i = \langle x, u_i \rangle\) 仅对标准正交基成立。若基向量未单位化，需除以 \(\|u_i\|^2\)，即 \(x_i = \langle x, u_i \rangle / \|u_i\|^2\)。

5. **忽视数值误差**：在计算机实现中，经典Gram-Schmidt可能因舍入误差产生非正交结果。例如，对接近线性相关的向量组，正交性可能严重退化，此时应使用Householder变换或SVD分解。
