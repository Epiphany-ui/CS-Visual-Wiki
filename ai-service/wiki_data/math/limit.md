# 极限

## 一、定义

极限（Limit）是数学分析的核心概念，描述了一个函数或数列在自变量无限趋近于某一点（或无穷大）时，其值无限趋近于某个确定常数的行为。从形式化定义看，数列极限的ε-N定义如下：

设{xₙ}为一数列，若存在常数a，使得∀ε>0，∃N∈ℕ⁺，当n>N时，恒有|xₙ-a|<ε，则称数列{xₙ}收敛于a，记作lim_{n→∞} xₙ = a。

函数极限的ε-δ定义：设f(x)在x₀的某去心邻域有定义，若∀ε>0，∃δ>0，当0<|x-x₀|<δ时，恒有|f(x)-A|<ε，则称lim_{x→x₀} f(x)=A。

在计算机科学中，极限常用于分析算法的渐进复杂度，如O(n)、Ω(n)等记号本质上都是极限概念的延伸。

## 二、核心原理

1. **无限逼近思想**：极限的核心在于“无限接近但未必到达”。数列或函数的值可以无限趋近于某个常数，但该常数可能永远无法被精确取到。例如数列1/n无限趋近于0，但任何有限项都不等于0。

2. **任意小误差控制**：ε-N和ε-δ定义的精髓在于“任意小”。对于任何预先给定的正数ε（无论多小），总能找到对应的N或δ，使得后续所有项或邻域内的所有点都满足误差小于ε。

3. **局部性质**：函数极限只关心x₀附近的行为，与x₀点本身是否有定义、函数值是多少完全无关。这解释了为什么可去间断点处极限仍然存在。

4. **唯一性与保号性**：若极限存在，则必唯一。若极限大于0，则存在某邻域内函数值也大于0（保号性），这是许多不等式证明的基础。

5. **四则运算法则**：若两个函数的极限都存在，则它们的和、差、积、商（分母极限不为0）的极限等于极限的相应运算，这为复杂极限计算提供了分解手段。

## 三、过程/示例

考虑数列xₙ = (n+1)/n，求lim_{n→∞} xₙ。

**步骤1：观察趋势**  
计算前几项：x₁=2，x₂=1.5，x₃≈1.333，x₄=1.25，x₅=1.2，x₁₀=1.1，x₁₀₀=1.01。可见数列值逐渐减小并趋近于1。

**步骤2：猜测极限值**  
将表达式改写：xₙ = 1 + 1/n。当n→∞时，1/n→0，因此猜测极限为1。

**步骤3：用ε-N定义验证**  
要证lim xₙ=1，即∀ε>0，∃N，当n>N时，|xₙ-1|<ε。  
计算|xₙ-1| = |(n+1)/n - 1| = |1/n| = 1/n。  
令1/n < ε，得n > 1/ε。取N = ⌈1/ε⌉，则当n>N时，必有1/n < ε。  
例如取ε=0.01，则N=100，当n>100时，|xₙ-1|<0.01成立。

**步骤4：确认结论**  
由ε的任意性，极限存在且等于1。该数列收敛于1。

## 四、复杂度/性质分析

**收敛速度**：数列xₙ=1/n的收敛速度为O(1/n)，属于慢速收敛。指数衰减如xₙ=qⁿ（|q|<1）的收敛速度为O(qⁿ)，收敛更快。

**稳定性**：极限运算对微小扰动敏感。若数列各项有界误差，极限值可能发生偏移。例如xₙ=1/n+δ，若δ为常数误差，则极限变为δ而非0。

**计算复杂度**：数值计算中，用迭代法求极限（如二分法求方程根）的时间复杂度通常为O(log(1/ε))，其中ε为精度要求。空间复杂度为O(1)。

**收敛性判定**：单调有界数列必收敛（实数完备性）；柯西收敛准则：数列收敛当且仅当∀ε>0，∃N，当m,n>N时，|xₘ-xₙ|<ε。这些性质在算法收敛性分析中至关重要。

**与渐进复杂度的关系**：算法分析中的大O记号f(n)=O(g(n))等价于limsup_{n→∞}|f(n)/g(n)|<∞，本质上是极限概念的推广。

## 五、常见误区

1. **混淆极限值与函数值**：认为lim_{x→x₀}f(x)必须等于f(x₀)。实际上极限与函数值无关，例如f(x)=sinx/x在x=0处无定义，但极限为1。连续函数才满足两者相等。

2. **误用极限运算法则**：认为lim(f(x)/g(x)) = lim f(x)/lim g(x)无条件成立。必须确保分母极限不为0且两个极限都存在。例如lim_{x→0} x/x²，分子极限0，分母极限0，不能直接套用。

3. **忽视无穷小阶的比较**：认为所有趋近于0的量等价。实际上sinx~x（当x→0），但1-cosx~x²/2，误用会导致错误结果。需要区分高阶、低阶、同阶无穷小。

4. **混淆数列极限与函数极限**：数列极限要求n→+∞，是离散的；函数极限可x→x₀或x→∞，是连续的。例如lim_{n→∞} sin(nπ)存在（=0），但lim_{x→∞} sin(πx)不存在。

5. **极限存在性的误判**：认为有界数列必有极限。反例：xₙ=(-1)ⁿ有界但不收敛。单调有界才是收敛的充分条件。同样，函数在某点有界也不保证极限存在。
