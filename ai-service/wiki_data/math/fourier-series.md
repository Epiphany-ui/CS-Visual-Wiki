# 傅里叶级数

## 一、定义

傅里叶级数（Fourier Series）是将周期函数分解为一系列正弦和余弦函数（或复指数函数）线性组合的数学工具。对于周期为 \(T\) 的函数 \(f(t)\)，其傅里叶级数展开式为：

\[
f(t) = \frac{a_0}{2} + \sum_{n=1}^{\infty} \left[ a_n \cos\left(\frac{2\pi n t}{T}\right) + b_n \sin\left(\frac{2\pi n t}{T}\right) \right]
\]

其中系数由以下积分计算：

\[
a_n = \frac{2}{T} \int_{0}^{T} f(t) \cos\left(\frac{2\pi n t}{T}\right) dt, \quad b_n = \frac{2}{T} \int_{0}^{T} f(t) \sin\left(\frac{2\pi n t}{T}\right) dt
\]

在计算机科学中，离散傅里叶级数（DFS）处理有限长序列，其核心公式为：

\[
X[k] = \sum_{n=0}^{N-1} x[n] e^{-j\frac{2\pi}{N}kn}, \quad k = 0,1,\dots,N-1
\]

## 二、核心原理

1. **正交基分解**：正弦和余弦函数族在区间 \([0,T]\) 上构成完备正交基，任意周期函数可唯一表示为这些基函数的加权和。正交性保证了系数计算的独立性。

2. **频率域表示**：傅里叶级数将时域信号映射到频域，\(a_n\) 和 \(b_n\) 分别表示各频率分量的余弦和正弦幅度。这种转换揭示了信号的频谱结构。

3. **收敛性条件**：狄利克雷条件要求函数在周期内只有有限个第一类间断点和有限个极值点。在连续点处级数收敛于函数值，在间断点处收敛于左右极限的平均值。

4. **帕塞瓦尔定理**：信号在时域的总能量等于频域各分量能量之和，即 \(\frac{1}{T}\int_0^T |f(t)|^2 dt = \frac{a_0^2}{4} + \frac{1}{2}\sum_{n=1}^\infty (a_n^2 + b_n^2)\)。

5. **吉布斯现象**：在函数的不连续点附近，傅里叶级数部分和会出现过冲，最大过冲量约为跳变幅度的9%，且不随项数增加而消失。

## 三、过程/示例

考虑周期为 \(2\pi\) 的方波函数：

\[
f(t) = \begin{cases} 
1, & 0 < t < \pi \\
-1, & \pi < t < 2\pi
\end{cases}
\]

**步骤1：计算直流分量**  
\(a_0 = \frac{1}{\pi} \int_0^{2\pi} f(t) dt = \frac{1}{\pi} \left( \int_0^\pi 1 dt + \int_\pi^{2\pi} (-1) dt \right) = 0\)

**步骤2：计算余弦系数**  
\(a_n = \frac{1}{\pi} \int_0^{2\pi} f(t) \cos(nt) dt = \frac{1}{\pi} \left( \int_0^\pi \cos(nt) dt - \int_\pi^{2\pi} \cos(nt) dt \right) = 0\)（奇函数在对称区间积分为零）

**步骤3：计算正弦系数**  
\(b_n = \frac{1}{\pi} \int_0^{2\pi} f(t) \sin(nt) dt = \frac{1}{\pi} \left( \int_0^\pi \sin(nt) dt - \int_\pi^{2\pi} \sin(nt) dt \right)\)

计算得：\(b_n = \frac{2}{n\pi} [1 - (-1)^n]\)，即：
- \(n\)为偶数时 \(b_n = 0\)
- \(n\)为奇数时 \(b_n = \frac{4}{n\pi}\)

**步骤4：写出级数展开**  
\(f(t) = \frac{4}{\pi} \left( \sin t + \frac{1}{3}\sin 3t + \frac{1}{5}\sin 5t + \cdots \right)\)

取前3项近似：\(f(t) \approx \frac{4}{\pi}(\sin t + \frac{1}{3}\sin 3t + \frac{1}{5}\sin 5t)\)

## 四、复杂度/性质分析

**时间复杂度**：
- 直接计算N点离散傅里叶级数：\(O(N^2)\)，每个\(X[k]\)需N次乘加运算
- 快速傅里叶变换（FFT）：\(O(N \log N)\)，利用对称性和周期性简化计算
- 最好/最坏情况相同，均为\(O(N \log N)\)（FFT）

**空间复杂度**：
- 直接计算：\(O(N)\)（存储输入和输出序列）
- FFT：\(O(N)\)（原地计算需额外\(O(1)\)辅助空间）

**收敛性质**：
- 连续点处：均匀收敛（满足狄利克雷条件时）
- 间断点处：收敛于左右极限平均值，收敛速度受函数光滑性影响
- 光滑函数：系数以\(O(1/n^k)\)衰减（k为连续可导阶数）

**稳定性**：
- 数值计算中，高频系数对舍入误差敏感
- 吉布斯现象导致有限项近似在间断点附近出现振荡

## 五、常见误区

1. **混淆傅里叶级数与傅里叶变换**：级数针对周期信号，变换针对非周期信号。级数得到离散频谱，变换得到连续频谱。

2. **忽视收敛条件**：并非所有周期函数都能展开为傅里叶级数，必须满足狄利克雷条件。例如无限间断点的函数可能不收敛。

3. **误认为部分和处处逼近原函数**：在间断点处，部分和收敛于平均值而非函数值；吉布斯现象导致有限项近似在间断点附近出现固定幅度的过冲。

4. **混淆连续与离散形式**：连续傅里叶级数（CFS）处理模拟信号，离散傅里叶级数（DFS）处理数字信号。DFS本质上是CFS在时域和频域的双重采样。

5. **忽略相位信息**：初学者常只关注幅度谱\(|X[k]|\)，但相位谱\(\angle X[k]\)同样重要，它决定了各频率分量的相对时间偏移，对信号重构至关重要。
