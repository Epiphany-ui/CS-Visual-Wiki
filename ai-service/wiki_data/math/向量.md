# 向量

## 一、定义

向量（Vector）是既有大小又有方向的量，是线性代数与计算机科学中最基本的数学对象之一。在数学上，一个 \( n \) 维向量通常表示为有序元组：

\[
\mathbf{v} = (v_1, v_2, \dots, v_n) \in \mathbb{R}^n
\]

在计算机科学中，向量常以数组或列表形式存储，例如 Python 中的 `[v1, v2, ..., vn]`。向量的核心操作包括加法、数乘、点积和范数计算，这些构成了机器学习、图形学和物理模拟等领域的运算基础。

## 二、核心原理

1. **几何意义**：向量可视为从原点出发到空间某点的有向线段。其长度（模）由欧几里得范数定义：\(\|\mathbf{v}\| = \sqrt{\sum_{i=1}^n v_i^2}\)，方向由各分量比例决定。

2. **线性组合**：向量空间的核心性质是封闭性——任意两个向量的和与标量乘法结果仍在该空间中。给定向量 \(\mathbf{a}, \mathbf{b}\) 和标量 \(\alpha, \beta\)，有 \(\alpha\mathbf{a} + \beta\mathbf{b}\) 仍为同维向量。

3. **点积与正交性**：点积 \(\mathbf{a} \cdot \mathbf{b} = \sum_{i=1}^n a_i b_i\) 衡量两向量的对齐程度。当点积为零时，两向量正交（垂直），这是 Gram-Schmidt 正交化和 PCA 等算法的基础。

4. **基与坐标**：任意 \(n\) 维向量可由一组基向量的线性组合唯一表示。标准基 \(\mathbf{e}_1, \mathbf{e}_2, \dots, \mathbf{e}_n\) 中，\(\mathbf{e}_i\) 仅第 \(i\) 分量为 1，其余为 0。

5. **向量范数**：除欧几里得范数外，还有 \(L_1\) 范数（曼哈顿距离）和 \(L_\infty\) 范数（切比雪夫距离），分别定义为 \(\|\mathbf{v}\|_1 = \sum |v_i|\) 和 \(\|\mathbf{v}\|_\infty = \max |v_i|\)，在正则化和误差分析中各有用途。

## 三、过程/示例

以三维向量 \(\mathbf{a} = (3, 4, 0)\) 和 \(\mathbf{b} = (1, -2, 2)\) 为例，演示基本运算：

**步骤 1：向量加法**
\[
\mathbf{a} + \mathbf{b} = (3+1,\; 4+(-2),\; 0+2) = (4, 2, 2)
\]

**步骤 2：标量乘法**
取 \(\lambda = 2\)：
\[
2\mathbf{a} = (2 \times 3,\; 2 \times 4,\; 2 \times 0) = (6, 8, 0)
\]

**步骤 3：点积计算**
\[
\mathbf{a} \cdot \mathbf{b} = 3 \times 1 + 4 \times (-2) + 0 \times 2 = 3 - 8 + 0 = -5
\]
负值表明两向量夹角大于 90°。

**步骤 4：范数计算**
\[
\|\mathbf{a}\| = \sqrt{3^2 + 4^2 + 0^2} = \sqrt{25} = 5
\]
\[
\|\mathbf{b}\| = \sqrt{1^2 + (-2)^2 + 2^2} = \sqrt{9} = 3
\]

**步骤 5：单位向量**
\[
\hat{\mathbf{a}} = \frac{\mathbf{a}}{\|\mathbf{a}\|} = \left(\frac{3}{5}, \frac{4}{5}, 0\right)
\]

## 四、复杂度/性质分析

- **时间复杂度**：
  - 向量加法/减法：\(O(n)\)，需遍历所有分量。
  - 点积：\(O(n)\)，一次乘加操作。
  - 欧几里得范数：\(O(n)\)，平方和后再开方。
  - 标量乘法：\(O(n)\)。
  - 所有操作均无最好/最坏情况差异，均为线性时间。

- **空间复杂度**：存储一个 \(n\) 维向量需要 \(O(n)\) 空间，通常以连续内存块实现（如 C 数组或 Python list）。

- **数值稳定性**：点积和范数计算可能因大数平方导致浮点溢出，实践中常使用分段计算或 Kahan 求和算法。

- **代数性质**：
  - 加法交换律：\(\mathbf{a} + \mathbf{b} = \mathbf{b} + \mathbf{a}\)
  - 加法结合律：\((\mathbf{a} + \mathbf{b}) + \mathbf{c} = \mathbf{a} + (\mathbf{b} + \mathbf{c})\)
  - 数乘分配律：\(\lambda(\mathbf{a} + \mathbf{b}) = \lambda\mathbf{a} + \lambda\mathbf{b}\)
  - 点积对称性：\(\mathbf{a} \cdot \mathbf{b} = \mathbf{b} \cdot \mathbf{a}\)

## 五、常见误区

1. **向量与标量的混淆**：初学者常将向量误认为普通数字。向量是有序分量集合，而标量是单一数值。例如 \(2\mathbf{v}\) 是数乘，\(\mathbf{v} + 2\) 在标准向量空间中无定义。

2. **点积结果误解**：点积返回标量而非向量，且不满足消去律——\(\mathbf{a} \cdot \mathbf{b} = \mathbf{a} \cdot \mathbf{c}\) 不能推出 \(\mathbf{b} = \mathbf{c}\)，因为可能存在与 \(\mathbf{a}\) 正交的分量差异。

3. **范数与维度的混淆**：向量的“大小”指模长而非维度。例如 \(\mathbf{v} = (0,0)\) 是二维向量，但其模长为 0（零向量），初学者易将“二维”误解为“长度为 2”。

4. **向量与点的混用**：在计算机图形学中，向量表示位移，点表示位置。两者运算规则不同：点减点得向量，点加向量得点，但点加点无几何意义。

5. **正交与线性无关的混淆**：正交向量必然线性无关，但线性无关向量不一定正交。例如 \(\{(1,0), (1,1)\}\) 线性无关却不正交，初学者常将两者等同。
