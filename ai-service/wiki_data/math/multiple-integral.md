# 多重积分

## 一、定义
多重积分是单变量定积分在多维空间上的推广，用于计算定义在$\mathbb{R}^n$区域上的函数对体积（面积）的累积效应。对于二元函数$f(x,y)$，二重积分记作$\iint_D f(x,y)\,dA$；对于三元函数$f(x,y,z)$，三重积分记作$\iiint_V f(x,y,z)\,dV$。其核心数学定义为黎曼和的极限：将区域$D$分割成$n$个小区域$\Delta A_i$，取样本点$(\xi_i,\eta_i)$，则
$$\iint_D f(x,y)\,dA = \lim_{\max \Delta A_i \to 0} \sum_{i=1}^n f(\xi_i,\eta_i)\Delta A_i$$
当该极限存在且与分割方式无关时，称函数在区域上可积。

## 二、核心原理
1. **累次积分思想**：将多重积分转化为多个单变量积分的嵌套计算。对于二重积分，先对$y$积分（视$x$为常数），再对$x$积分，即$\iint_D f(x,y)dA = \int_a^b \left[\int_{y_1(x)}^{y_2(x)} f(x,y)dy\right]dx$。这一思想基于富比尼定理，要求积分区域可表示为简单$xy$型或$yx$型区域。

2. **区域分解与积分限确定**：将复杂区域分解为若干简单子区域，每个子区域在某一坐标轴方向上的投影是区间，另一方向边界由函数描述。例如$xy$型区域：$D=\{(x,y)|a\le x\le b,\; y_1(x)\le y\le y_2(x)\}$。积分限的确定遵循“先内后外”原则，内层积分限依赖于外层变量。

3. **坐标变换与雅可比行列式**：当区域形状复杂或函数具有对称性时，通过变量替换简化计算。变换$(x,y)=\mathbf{T}(u,v)$后，面积元变换为$dxdy = |J|\,dudv$，其中$J=\frac{\partial(x,y)}{\partial(u,v)}$为雅可比行列式。典型变换包括极坐标（$dxdy = r\,drd\theta$）、柱坐标（$dV = r\,drd\theta dz$）和球坐标（$dV = \rho^2\sin\phi\,d\rho d\phi d\theta$）。

4. **积分次序交换**：根据区域描述和函数特性，灵活选择先对$x$还是先对$y$积分。交换次序需重新描述区域边界，有时能显著简化计算，例如将不可积的初等函数积分变为可积形式。

## 三、过程/示例
计算二重积分$\iint_D (x+y)\,dA$，其中$D$是由$y=x^2$和$y=x$围成的区域。

**步骤1：确定区域描述**  
两曲线交点：$x^2=x \Rightarrow x=0,1$。区域可描述为$xy$型：$0\le x\le 1$，$x^2\le y\le x$。

**步骤2：化为累次积分**  
$$\iint_D (x+y)dA = \int_{x=0}^1 \left[\int_{y=x^2}^x (x+y)dy\right]dx$$

**步骤3：计算内层积分**  
对$y$积分，视$x$为常数：  
$$\int_{y=x^2}^x (x+y)dy = \left[xy + \frac{y^2}{2}\right]_{y=x^2}^x = \left(x\cdot x + \frac{x^2}{2}\right) - \left(x\cdot x^2 + \frac{x^4}{2}\right) = \frac{3x^2}{2} - x^3 - \frac{x^4}{2}$$

**步骤4：计算外层积分**  
$$\int_0^1 \left(\frac{3x^2}{2} - x^3 - \frac{x^4}{2}\right)dx = \left[\frac{x^3}{2} - \frac{x^4}{4} - \frac{x^5}{10}\right]_0^1 = \frac{1}{2} - \frac{1}{4} - \frac{1}{10} = \frac{3}{20}$$

因此$\iint_D (x+y)dA = \frac{3}{20}$。

## 四、复杂度/性质分析
1. **计算复杂度**：多重积分的数值计算（如蒙特卡洛方法）时间复杂度为$O(N)$，其中$N$为采样点数，但收敛速度慢（误差$O(1/\sqrt{N})$）。确定性数值方法（如复合辛普森公式）在$d$维空间中的网格点数为$O(n^d)$，计算复杂度呈指数增长，称为“维数灾难”。

2. **可积性条件**：有界闭区域上的连续函数必可积；具有有限个间断点的有界函数仍可积（黎曼可积）。无界函数或无界区域需考虑反常多重积分，收敛性判定依赖于比较判别法和绝对收敛性。

3. **对称性简化**：若积分区域关于坐标轴对称且函数具有奇偶性，可大幅简化。例如区域关于$x$轴对称，$f(x,-y)=-f(x,y)$时积分值为0；$f(x,-y)=f(x,y)$时积分值等于上半区域积分的2倍。

4. **收敛性与稳定性**：数值计算中，蒙特卡洛方法对高维积分收敛稳定但精度低；确定性方法在低维（$d\le 3$）时精度高，但高维时不稳定。解析计算中，积分次序选择不当可能导致不可积的初等函数形式，需谨慎处理。

## 五、常见误区
1. **积分限顺序混淆**：误将外层积分变量写在前面。正确写法是$\int_{x=a}^b \int_{y=y_1(x)}^{y_2(x)} f(x,y)dy\,dx$，内层积分限可依赖外层变量，但外层积分限必须是常数。

2. **雅可比行列式取绝对值**：坐标变换时忘记乘以$|J|$，或误用$J$的符号。面积元变换公式为$dxdy = |J|\,dudv$，雅可比行列式可能为负，但面积元必须为正。

3. **区域描述错误**：将非简单区域强行表示为$xy$型或$yx$型，导致积分限函数不唯一。例如环形区域需分割为多个子区域分别积分，或使用极坐标统一处理。

4. **混淆重积分与曲线/曲面积分**：多重积分是对区域内部的体积分，而曲线积分（第一类）和曲面积分（第一类）分别是对曲线弧长和曲面面积的积分，积分区域维度不同，物理意义也不同。

5. **忽略积分次序可行性**：某些函数在一种积分次序下无法用初等函数表示，但交换次序后可积。例如$\int_0^1\int_y^1 e^{x^2}dx\,dy$直接积分困难，交换次序后变为$\int_0^1\int_0^x e^{x^2}dy\,dx$，可轻松计算。
