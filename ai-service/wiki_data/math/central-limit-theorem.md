# 中心极限定理

## 一、定义
中心极限定理（Central Limit Theorem, CLT）是概率论中描述独立随机变量之和的极限分布趋于正态分布的一组定理。其核心结论是：对于任意具有有限均值μ和有限方差σ²的独立同分布随机变量序列X₁, X₂, …, Xₙ，当样本容量n足够大时，其样本均值\(\bar{X}_n = \frac{1}{n}\sum_{i=1}^{n}X_i\)的标准化统计量近似服从标准正态分布。数学表达式为：

\[
\frac{\bar{X}_n - \mu}{\sigma / \sqrt{n}} \xrightarrow{d} N(0, 1) \quad \text{当 } n \to \infty
\]

其中\(\xrightarrow{d}\)表示依分布收敛，N(0,1)为标准正态分布。该定理不要求原始分布本身是正态分布，仅要求方差有限。

## 二、核心原理
1. **叠加效应**：独立随机变量的和或均值的分布，随着变量个数增加，逐渐趋近于正态分布。这一性质源于大量微小随机扰动的累积效应，与原始分布的具体形态无关。

2. **标准化机制**：通过减去均值并除以标准差，将任意分布的样本均值转化为均值为0、方差为1的标准化变量。标准化消除了原始分布的尺度差异，使得不同分布的比较成为可能。

3. **收敛速度**：收敛速度取决于原始分布的偏度和峰度。对称分布（如均匀分布）收敛较快，偏斜分布（如指数分布）需要更大的样本量。Berry-Esseen定理给出了收敛速度的上界：\(O(1/\sqrt{n})\)。

4. **样本量要求**：理论上n→∞，实践中通常认为n≥30即可获得较好的正态近似。但对于高度偏斜或重尾分布，可能需要更大的样本量（如n≥100）。

5. **独立性假设**：定理要求变量间相互独立。若存在相关性，则需使用更一般的Lindeberg-Feller中心极限定理，其条件更为宽松。

## 三、过程/示例
考虑一个简单的离散均匀分布：掷一枚公平的六面骰子，每次掷出的点数X服从{1,2,3,4,5,6}上的均匀分布，均值μ=3.5，方差σ²≈2.9167。

**步骤1：单次抽样**  
单次掷骰子的结果分布是均匀的，每个点数概率均为1/6，直方图呈平坦矩形。

**步骤2：两次掷骰子之和**  
计算所有可能的和值（2到12）及其概率。例如，和为7的概率为6/36=1/6，和为2或12的概率均为1/36。此时分布已呈现中间高、两端低的三角形形状。

**步骤3：三次掷骰子之和**  
和值范围3到18，概率分布进一步向中心集中。例如，和为10或11的概率最高，约为12.5%。

**步骤4：三十次掷骰子均值**  
模拟30次掷骰子，计算样本均值\(\bar{X}_{30}\)。重复此过程10000次，绘制均值的直方图。结果显示：直方图近似钟形曲线，中心位于3.5附近，标准差约为\(\sigma/\sqrt{30} \approx 0.312\)。

**步骤5：标准化验证**  
对每个样本均值计算z-score：\(z = (\bar{X}_{30} - 3.5) / 0.312\)。这些z-score的分布与标准正态分布N(0,1)的累积分布函数高度吻合，Kolmogorov-Smirnov检验的p值通常大于0.05，表明无法拒绝正态性假设。

## 四、复杂度/性质分析
- **收敛速度**：误差以\(O(1/\sqrt{n})\)的速率衰减。具体地，对于均值为μ、方差为σ²的分布，标准化样本均值的分布函数与标准正态分布函数的最大绝对误差不超过\(C \cdot \rho / \sigma^3 \sqrt{n}\)，其中C为常数（约0.5），ρ为三阶绝对矩。

- **样本量要求**：  
  - 对称单峰分布（如均匀分布）：n≥10即可  
  - 轻度偏斜分布（如二项分布p≈0.5）：n≥30  
  - 高度偏斜分布（如指数分布）：n≥100  
  - 重尾分布（如t分布自由度≤3）：可能不适用（方差无限）

- **局限性**：  
  - 要求方差有限，柯西分布等重尾分布不适用  
  - 独立同分布假设在实际应用中常被违反  
  - 收敛是分布上的，而非逐点收敛

- **多维推广**：多维中心极限定理指出，随机向量的和收敛于多元正态分布，协方差矩阵由各分量的协方差决定。

## 五、常见误区
1. **原始数据需正态分布**：CLT描述的是样本均值的分布，而非原始数据的分布。原始数据可以是任意分布，只要方差有限。

2. **n≥30的绝对标准**：30是经验阈值，并非普适。对于高度偏斜分布（如对数正态分布），可能需要n≥100才能获得可靠近似。

3. **小样本适用性**：当n很小时（如n<5），样本均值的分布可能严重偏离正态，此时应使用精确分布或t分布。

4. **与大数定律混淆**：大数定律（LLN）保证样本均值收敛于总体均值（点估计），而CLT描述的是样本均值围绕总体均值的波动分布（区间估计）。LLN是CLT的基础，但两者解决的问题不同。

5. **相关数据的误用**：CLT要求独立性。对于时间序列等自相关数据，直接应用CLT会低估方差，导致错误的置信区间。此时需使用自相关稳健的标准误估计。
