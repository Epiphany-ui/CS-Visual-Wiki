# 概率论

## 一、定义

概率论是研究随机现象数量规律的数学分支。其核心是通过数学模型量化不确定性，为随机事件赋予0到1之间的数值（概率），并研究多个随机事件之间的关联与演化规律。概率论为统计学、机器学习、信息论、随机算法等计算机科学领域提供了理论基础。

**核心数学定义**：概率空间由三元组 $(\Omega, \mathcal{F}, P)$ 构成，其中：
- $\Omega$ 是样本空间，包含所有可能的基本结果。
- $\mathcal{F}$ 是事件域，是 $\Omega$ 的子集构成的 $\sigma$-代数。
- $P$ 是概率测度，满足：$P(\Omega)=1$，且对互不相容事件序列 $\{A_i\}$ 有可列可加性 $P(\bigcup_{i=1}^{\infty} A_i) = \sum_{i=1}^{\infty} P(A_i)$。

**核心公式**（条件概率与全概率公式）：
$$P(A|B) = \frac{P(A \cap B)}{P(B)}, \quad P(B) > 0$$
$$P(A) = \sum_{i=1}^{n} P(A|B_i)P(B_i)$$
其中 $\{B_i\}$ 是样本空间的一个完备划分。

## 二、核心原理

1. **概率的公理化体系**：概率不是经验的模糊感觉，而是满足三条基本公理的数学函数。非负性（$P(A) \ge 0$）、规范性（$P(\Omega)=1$）、可加性（互斥事件概率相加）构成了所有概率推导的基石。

2. **随机变量与分布函数**：随机变量是将样本空间映射到实数的函数。其概率分布完全由累积分布函数 $F(x)=P(X \le x)$ 描述。离散型用概率质量函数（PMF），连续型用概率密度函数（PDF）。

3. **大数定律**：当试验次数足够多时，样本均值依概率收敛于期望值。这是蒙特卡洛方法和统计估计的理论依据——随机模拟的精度随样本量增加而提高。

4. **中心极限定理**：独立同分布随机变量的和（或均值）近似服从正态分布，无论原始分布形态如何。这解释了正态分布在自然界和工程中的普遍性，也是置信区间和假设检验的基础。

5. **条件独立与马尔可夫性**：事件 $A$ 和 $B$ 在给定 $C$ 下条件独立当且仅当 $P(A \cap B|C) = P(A|C)P(B|C)$。这一性质是贝叶斯网络、隐马尔可夫模型等概率图模型的核心假设。

## 三、过程/示例

**问题**：一个不透明的袋中有3个红球和2个蓝球。不放回地连续抽取两次，求第二次抽到红球的概率。

**步骤1：定义事件**
- 设 $A$ 为第一次抽到红球，$\bar{A}$ 为第一次抽到蓝球。
- 设 $B$ 为第二次抽到红球。

**步骤2：计算先验概率**
- $P(A) = \frac{3}{5} = 0.6$
- $P(\bar{A}) = \frac{2}{5} = 0.4$

**步骤3：计算条件概率**
- 若第一次抽到红球（剩余2红2蓝）：$P(B|A) = \frac{2}{4} = 0.5$
- 若第一次抽到蓝球（剩余3红1蓝）：$P(B|\bar{A}) = \frac{3}{4} = 0.75$

**步骤4：应用全概率公式**
$$P(B) = P(B|A)P(A) + P(B|\bar{A})P(\bar{A}) = 0.5 \times 0.6 + 0.75 \times 0.4 = 0.3 + 0.3 = 0.6$$

**结果**：第二次抽到红球的概率为0.6，恰好等于第一次抽到红球的概率。这体现了无放回抽样中“对称性”的直观结论。

## 四、复杂度/性质分析

**计算复杂度**：
- 直接计算离散概率空间：$O(|\Omega|)$，其中 $|\Omega|$ 是样本空间大小。对于 $n$ 个独立二元变量，$|\Omega| = 2^n$，呈指数增长。
- 贝叶斯推断（精确）：$O(2^n)$ 或更高，NP-hard问题。
- 蒙特卡洛近似：$O(N)$，$N$ 为采样次数，精度随 $O(1/\sqrt{N})$ 收敛。

**关键性质**：
- **收敛性**：大数定律保证样本均值收敛于期望；中心极限定理给出收敛速率。
- **稳定性**：概率测度对微小扰动敏感度由 Lipschitz 常数控制，在机器学习中体现为泛化误差界。
- **可计算性**：精确概率推理在一般图模型中是 #P-complete 的，需依赖近似算法（MCMC、变分推断）。
- **信息论联系**：概率分布的不确定性由熵 $H(X) = -\sum p(x)\log p(x)$ 度量，互信息 $I(X;Y)$ 刻画变量间的依赖强度。

## 五、常见误区

1. **混淆“概率为0”与“不可能事件”**：连续型随机变量取任意特定值的概率为0，但该事件并非不可能。例如，从 $[0,1]$ 均匀分布中恰好抽到0.5的概率为0，但抽到0.5是可能发生的。

2. **赌徒谬误**：认为独立随机事件的历史会影响未来概率。例如，连续5次抛硬币得到正面后，误以为下一次反面的概率更大。实际上每次抛掷仍为独立0.5概率。

3. **条件概率的颠倒**：将 $P(A|B)$ 与 $P(B|A)$ 混淆。例如，$P(患病|检测阳性)$ 与 $P(检测阳性|患病)$ 完全不同，前者需通过贝叶斯公式计算，通常远小于后者（受基础发病率影响）。

4. **忽略条件独立性假设**：朴素贝叶斯分类器假设特征在给定类别下条件独立，但实际数据往往违反此假设。强行使用会导致概率估计偏差，但分类决策边界可能仍然可用。

5. **期望的线性性误解**：期望的线性性 $E[X+Y]=E[X]+E[Y]$ 对任意随机变量成立（无需独立），但 $E[XY]=E[X]E[Y]$ 仅在独立时成立。初学者常错误地将乘积期望也视为线性可拆。
