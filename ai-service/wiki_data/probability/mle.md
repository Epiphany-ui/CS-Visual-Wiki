# 最大似然估计

## 一、定义

最大似然估计（Maximum Likelihood Estimation, MLE）是一种在给定观测数据下，通过最大化似然函数来估计概率模型参数的方法。其核心思想是：选择使当前观测数据出现概率最大的参数值作为估计值。

设随机变量 \(X\) 的概率密度函数（或概率质量函数）为 \(f(x;\theta)\)，其中 \(\theta\) 为待估参数。给定独立同分布的观测样本 \(x_1, x_2, \ldots, x_n\)，似然函数定义为：

\[
L(\theta) = \prod_{i=1}^{n} f(x_i;\theta)
\]

最大似然估计的目标是找到参数 \(\hat{\theta}\)，使得：

\[
\hat{\theta} = \arg\max_{\theta} L(\theta)
\]

由于连乘运算容易导致数值下溢，通常对似然函数取对数，得到对数似然函数：

\[
\ell(\theta) = \sum_{i=1}^{n} \ln f(x_i;\theta)
\]

则最大似然估计等价于：

\[
\hat{\theta} = \arg\max_{\theta} \ell(\theta)
\]

## 二、核心原理

1. **似然与概率的区分**：概率是在已知参数下预测数据出现的可能性；似然则是在已知数据下，参数取不同值时数据出现的相对可能性。MLE将参数视为变量，寻找使数据“最合理”的参数值。

2. **独立同分布假设**：MLE通常假设样本独立同分布，这使得联合概率可以分解为各样本概率的乘积，简化计算。

3. **对数变换的数学便利**：取对数将乘积转化为求和，便于求导；同时对数函数是单调递增函数，不改变极值点位置。

4. **求解方法**：对于可导的似然函数，通常通过求解似然方程 \(\frac{\partial \ell(\theta)}{\partial \theta} = 0\) 得到极值点；对于复杂模型，可采用梯度下降、牛顿法等数值优化方法。

5. **渐近性质**：在正则条件下，MLE具有一致性（随样本量增大收敛到真实值）、渐近正态性和渐近有效性（达到克拉美-罗下界）。

## 三、过程/示例

以估计正态分布 \(N(\mu, \sigma^2)\) 的参数为例，观测数据为：\(x = [1.2, 0.8, 1.5, 0.9, 1.1]\)。

**步骤1：写出似然函数**

正态分布的概率密度函数为：
\[
f(x;\mu,\sigma^2) = \frac{1}{\sqrt{2\pi\sigma^2}} \exp\left(-\frac{(x-\mu)^2}{2\sigma^2}\right)
\]

似然函数：
\[
L(\mu,\sigma^2) = \prod_{i=1}^{5} \frac{1}{\sqrt{2\pi\sigma^2}} \exp\left(-\frac{(x_i-\mu)^2}{2\sigma^2}\right)
\]

**步骤2：取对数似然**

\[
\ell(\mu,\sigma^2) = -\frac{5}{2}\ln(2\pi) - \frac{5}{2}\ln(\sigma^2) - \frac{1}{2\sigma^2}\sum_{i=1}^{5}(x_i-\mu)^2
\]

**步骤3：对 \(\mu\) 求偏导并令为0**

\[
\frac{\partial \ell}{\partial \mu} = \frac{1}{\sigma^2}\sum_{i=1}^{5}(x_i-\mu) = 0
\]
解得：\(\hat{\mu} = \frac{1}{5}\sum_{i=1}^{5} x_i = \frac{1.2+0.8+1.5+0.9+1.1}{5} = 1.1\)

**步骤4：对 \(\sigma^2\) 求偏导并令为0**

\[
\frac{\partial \ell}{\partial \sigma^2} = -\frac{5}{2\sigma^2} + \frac{1}{2\sigma^4}\sum_{i=1}^{5}(x_i-\mu)^2 = 0
\]
解得：\(\hat{\sigma}^2 = \frac{1}{5}\sum_{i=1}^{5}(x_i-\hat{\mu})^2 = \frac{(0.1)^2+(-0.3)^2+(0.4)^2+(-0.2)^2+0^2}{5} = 0.06\)

因此，MLE估计结果为 \(\hat{\mu}=1.1\)，\(\hat{\sigma}^2=0.06\)。

## 四、复杂度/性质分析

**时间复杂度**：
- 解析解情况：\(O(n)\)，只需一次遍历计算统计量
- 数值优化情况：每次迭代 \(O(n)\)，迭代次数取决于算法收敛速度，通常为 \(O(kn)\)，其中 \(k\) 为迭代次数

**空间复杂度**：\(O(n)\) 用于存储观测数据，或 \(O(1)\) 若使用在线更新

**关键性质**：
1. **一致性**：当样本量 \(n \to \infty\) 时，\(\hat{\theta} \xrightarrow{p} \theta_0\)（真实参数）
2. **渐近正态性**：\(\sqrt{n}(\hat{\theta} - \theta_0) \xrightarrow{d} N(0, I^{-1}(\theta_0))\)，其中 \(I(\theta)\) 为Fisher信息量
3. **不变性**：若 \(\hat{\theta}\) 是 \(\theta\) 的MLE，则对任意函数 \(g(\cdot)\)，\(g(\hat{\theta})\) 是 \(g(\theta)\) 的MLE
4. **充分性**：MLE依赖于充分统计量，即它只通过充分统计量利用数据信息

**收敛性**：在正则条件下，MLE几乎必然收敛到真实参数值，收敛速度为 \(O(1/\sqrt{n})\)。

## 五、常见误区

1. **混淆似然与概率**：初学者常将似然函数理解为“参数为某值的概率”。实际上，似然不是概率分布，其积分不一定为1，不能直接解释为概率。

2. **忽视二阶条件**：求解似然方程得到驻点后，需验证二阶导数是否为负（即海森矩阵负定），否则可能是极小值或鞍点。

3. **样本量不足时的偏差**：MLE在小样本下可能有偏。例如正态分布方差估计中，\(\hat{\sigma}^2_{MLE} = \frac{1}{n}\sum(x_i-\bar{x})^2\) 是有偏的，无偏估计为 \(\frac{1}{n-1}\sum(x_i-\bar{x})^2\)。

4. **与最大后验估计（MAP）混淆**：MLE只考虑似然函数，而MAP引入参数的先验分布，最大化后验概率 \(P(\theta|X) \propto P(X|\theta)P(\theta)\)。当先验为均匀分布时，MAP退化为MLE。

5. **误认为MLE总是存在且唯一**：某些模型（如混合模型）的似然函数可能无界或存在多个局部极大值，此时MLE可能不存在或不唯一，需要正则化或使用其他估计方法。
