# 假设检验

## 一、定义
假设检验（Hypothesis Testing）是数理统计学中一种基于样本数据对总体参数或分布形式做出推断的决策方法。其核心思想是：预先提出一个关于总体的假设（称为原假设 \(H_0\)），然后构造一个检验统计量，并计算在 \(H_0\) 成立条件下该统计量出现当前观测值或更极端值的概率（即 \(p\) 值）。若 \(p\) 值小于预先设定的显著性水平 \(\alpha\)，则拒绝 \(H_0\)，否则不拒绝 \(H_0\)。

核心数学公式（以单样本 \(z\) 检验为例）：
\[
z = \frac{\bar{x} - \mu_0}{\sigma / \sqrt{n}}
\]
其中 \(\bar{x}\) 为样本均值，\(\mu_0\) 为原假设下的总体均值，\(\sigma\) 为总体标准差，\(n\) 为样本容量。根据标准正态分布计算 \(p\) 值：\(p = P(|Z| \geq |z|)\)。

## 二、核心原理
1. **反证法思想**：假设检验借鉴了数学中的反证法逻辑——先假定原假设为真，然后观察样本数据是否与这一假定产生“显著矛盾”。若样本结果出现的概率极低（低于 \(\alpha\)），则认为原假设不成立。
2. **小概率事件原理**：小概率事件在一次试验中几乎不会发生。若在原假设成立的前提下，观测到的小概率事件竟然发生了，则我们有理由怀疑原假设的正确性。
3. **两类错误权衡**：第一类错误（弃真错误）——原假设为真但被拒绝，概率记为 \(\alpha\)；第二类错误（取伪错误）——原假设为假但未被拒绝，概率记为 \(\beta\)。两者此消彼长，实际应用中需根据问题严重性设定 \(\alpha\)（通常取 0.05 或 0.01）。
4. **检验统计量的构造**：检验统计量是连接样本与假设的桥梁，其分布必须在原假设下完全已知（或可近似）。常见统计量包括 \(z\) 统计量、\(t\) 统计量、\(\chi^2\) 统计量、\(F\) 统计量等。
5. **拒绝域与显著性水平**：拒绝域是使得原假设被拒绝的样本观测值集合，其边界由显著性水平 \(\alpha\) 决定。双侧检验的拒绝域位于分布两端，单侧检验位于一端。

## 三、过程/示例
**问题**：某饮料厂宣称其瓶装饮料净含量均值为 500ml。质检员随机抽取 9 瓶，测得净含量（ml）为：498, 502, 497, 501, 499, 500, 496, 503, 498。已知总体标准差 \(\sigma = 2\)ml，显著性水平 \(\alpha = 0.05\)。问是否应怀疑厂家的宣称？

**步骤 1：建立假设**
- 原假设 \(H_0: \mu = 500\)
- 备择假设 \(H_1: \mu \neq 500\)（双侧检验）

**步骤 2：选择检验统计量**
由于总体方差已知且样本量较小，使用 \(z\) 检验：
\[
z = \frac{\bar{x} - 500}{2 / \sqrt{9}}
\]

**步骤 3：计算样本统计量**
样本均值 \(\bar{x} = \frac{498+502+497+501+499+500+496+503+498}{9} = \frac{4494}{9} = 499.333\)
\[
z = \frac{499.333 - 500}{2 / 3} = \frac{-0.667}{0.667} = -1.0
\]

**步骤 4：确定拒绝域**
查标准正态分布表，\(\alpha = 0.05\) 的双侧检验临界值为 \(z_{\alpha/2} = 1.96\)。拒绝域为 \(|z| \geq 1.96\)。

**步骤 5：做出决策**
\(|z| = 1.0 < 1.96\)，未落入拒绝域，因此不拒绝原假设。结论：在 0.05 显著性水平下，没有充分证据表明净含量均值不等于 500ml。

## 四、复杂度/性质分析
1. **时间复杂度**：假设检验的计算复杂度主要取决于样本量 \(n\) 和统计量计算方式。对于 \(z\) 检验和 \(t\) 检验，计算均值和标准差的时间复杂度为 \(O(n)\)；对于非参数检验（如 Wilcoxon 秩和检验），排序步骤带来 \(O(n \log n)\) 的复杂度。
2. **空间复杂度**：基本假设检验只需存储样本数据，空间复杂度为 \(O(n)\)。某些重抽样方法（如 Bootstrap 检验）需要存储多次重抽样结果，空间需求可达 \(O(B \cdot n)\)，其中 \(B\) 为重抽样次数。
3. **收敛性**：根据中心极限定理，当样本量 \(n \to \infty\) 时，样本均值的分布趋近正态分布，因此基于正态近似的检验（如 \(z\) 检验）具有渐近正确性。对于大样本，\(t\) 检验与 \(z\) 检验趋于一致。
4. **检验功效**：功效函数 \(1 - \beta\) 描述了当备择假设为真时正确拒绝原假设的概率。功效随样本量增大、效应量增大或显著性水平放宽而提高。
5. **稳健性**：\(t\) 检验对正态性假设的偏离具有一定稳健性，尤其在样本量较大时；但严重偏态或存在极端异常值时，非参数检验（如 Mann-Whitney \(U\) 检验）更为稳健。

## 五、常见误区
1. **将“不拒绝 \(H_0\)”等同于“接受 \(H_0\)”**：不拒绝原假设仅意味着当前数据不足以推翻它，并不证明原假设为真。这类似于法律上的“证据不足，无罪推定”，而非“证明清白”。
2. **混淆统计显著性与实际显著性**：当样本量极大时，微小的差异也可能达到统计显著（\(p < 0.05\)），但这种差异在实际中可能毫无意义。应始终结合效应量（如 Cohen's \(d\)）进行判断。
3. **误解 \(p\) 值的含义**：\(p\) 值不是原假设为真的概率，也不是犯第一类错误的概率。它是在原假设成立下，观测到当前或更极端结果的概率。\(P(H_0 | \text{data}) \neq p\)。
4. **多重比较问题**：对同一组数据进行多次假设检验而不调整显著性水平，会极大增加犯第一类错误的概率。例如，进行 20 次独立检验，即使所有原假设都为真，期望也会出现 1 次显著结果（\(\alpha = 0.05\)）。常用校正方法包括 Bonferroni 校正、FDR 控制等。
5. **忽视检验的前提假设**：每种检验方法都有其适用条件。例如，独立样本 \(t\) 检验要求数据独立、正态分布且方差齐性；若违反这些假设，检验结果可能不可靠。使用前应进行正态性检验（如 Shapiro-Wilk 检验）和方差齐性检验（如 Levene 检验）。
