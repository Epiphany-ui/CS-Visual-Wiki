# 贝叶斯定理

## 一、定义

贝叶斯定理（Bayes’ Theorem）是概率论中描述两个条件概率之间关系的核心定理。它提供了一种在已知先验概率和似然函数的情况下，计算后验概率的方法。其核心数学公式为：

$$P(A|B) = \frac{P(B|A) \cdot P(A)}{P(B)}$$

其中：
- \(P(A|B)\) 是给定事件 B 发生后事件 A 的后验概率。
- \(P(A)\) 是事件 A 的先验概率。
- \(P(B|A)\) 是给定事件 A 发生后事件 B 的似然概率。
- \(P(B)\) 是事件 B 的边缘概率（归一化常数），通常通过全概率公式计算：\(P(B) = \sum_i P(B|A_i) P(A_i)\)。

在机器学习中，该定理常以分类器形式出现，其伪代码逻辑可表示为：

```
输入：先验 P(A), 似然 P(B|A), 边缘概率 P(B)
输出：后验 P(A|B)
步骤：
1. 计算分子 = P(B|A) * P(A)
2. 计算分母 = P(B)
3. 返回 分子 / 分母
```

## 二、核心原理

1. **逆向概率推理**：贝叶斯定理的核心思想是从“结果”反推“原因”。传统频率学派关注“已知原因下结果的概率”，而贝叶斯学派关注“观察到结果后原因的可能性”，实现了从数据到假设的逆向更新。

2. **先验与后验的迭代更新**：先验概率 \(P(A)\) 代表在获得新证据前对假设的初始信念。当新证据 B 出现后，通过似然 \(P(B|A)\) 调整信念，得到后验概率 \(P(A|B)\)。后验可作为下一轮推理的先验，形成持续学习闭环。

3. **归一化与证据权衡**：分母 \(P(B)\) 作为归一化因子，确保所有可能假设的后验概率之和为 1。它衡量了证据 B 在所有假设下的总体出现概率，起到“证据强度”的校准作用。

4. **与条件概率的对称性**：贝叶斯定理揭示了 \(P(A|B)\) 与 \(P(B|A)\) 之间的数学关系，二者通常不相等，但通过先验和边缘概率可相互转换。这一性质在诊断测试、垃圾邮件过滤等场景中至关重要。

## 三、过程/示例

**问题**：某疾病在人群中的患病率为 1%（先验）。检测方法的灵敏度（患病者检测为阳性的概率）为 99%，特异度（健康者检测为阴性的概率）为 98%。若某人检测结果为阳性，求其真正患病的概率。

**步骤**：
1. **定义事件**：
   - A：患病
   - B：检测阳性
2. **已知概率**：
   - \(P(A) = 0.01\)
   - \(P(\neg A) = 0.99\)
   - \(P(B|A) = 0.99\)
   - \(P(B|\neg A) = 1 - 0.98 = 0.02\)
3. **计算边缘概率 \(P(B)\)**：
   \[
   P(B) = P(B|A)P(A) + P(B|\neg A)P(\neg A) = 0.99 \times 0.01 + 0.02 \times 0.99 = 0.0099 + 0.0198 = 0.0297
   \]
4. **应用贝叶斯定理**：
   \[
   P(A|B) = \frac{0.99 \times 0.01}{0.0297} = \frac{0.0099}{0.0297} \approx 0.3333
   \]

**结果**：即使检测阳性，真正患病的概率仅为约 33.3%，远低于直觉判断的 99%。这是因为健康人群基数大，假阳性数量不可忽视。

## 四、复杂度/性质分析

- **时间复杂度**：单次贝叶斯计算为 \(O(1)\)（仅涉及乘除法）。但在朴素贝叶斯分类器中，若特征维度为 \(d\)，类别数为 \(k\)，训练阶段计算先验和条件概率的时间复杂度为 \(O(n \cdot d)\)（n 为样本数），预测阶段为 \(O(k \cdot d)\)。
- **空间复杂度**：存储先验和条件概率表所需空间为 \(O(k \cdot d \cdot v)\)，其中 v 为每个特征的可能取值数。对于连续特征，需存储高斯分布参数，空间为 \(O(k \cdot d)\)。
- **收敛性**：贝叶斯估计本身是精确的，不涉及迭代收敛问题。但在贝叶斯推断的近似方法（如 MCMC）中，收敛性取决于采样算法和链长，通常需诊断如 Gelman-Rubin 统计量。
- **稳定性**：当先验概率极小时，后验对似然估计的误差敏感。若训练数据稀疏，条件概率估计可能不稳定，常用拉普拉斯平滑（加一平滑）缓解。
- **可解释性**：贝叶斯定理提供概率化输出，天然支持不确定性量化，这是其相比点估计方法的显著优势。

## 五、常见误区

1. **混淆先验与后验**：初学者常将先验概率 \(P(A)\) 直接当作最终结论，忽略新证据的更新作用。例如，认为“疾病患病率 1%”即个人患病概率，而忽视检测结果的影响。

2. **忽视基率谬误**：在示例中，许多人忽略疾病低患病率（1%）而过度依赖检测灵敏度（99%），导致高估后验概率。这是贝叶斯推理中最常见的认知偏差。

3. **误认为 \(P(A|B) = P(B|A)\)**：这是条件概率的对称性陷阱。例如，将“检测阳性时患病概率”等同于“患病时检测阳性概率”，二者数值可能相差巨大。

4. **忽略归一化分母**：部分学习者只计算分子 \(P(B|A)P(A)\)，忘记除以 \(P(B)\)，导致概率值未归一化，无法正确反映后验概率的绝对大小。

5. **将贝叶斯定理等同于朴素贝叶斯**：贝叶斯定理是通用数学工具，而朴素贝叶斯是假设特征条件独立的分类算法。后者是前者的简化应用，并非定理本身。
