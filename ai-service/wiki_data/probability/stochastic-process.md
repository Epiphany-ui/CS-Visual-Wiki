# 随机过程

## 一、定义

随机过程（Stochastic Process）是描述随机现象随时间演化的数学模型。严格定义：设概率空间 $(\Omega, \mathcal{F}, P)$，参数集 $T$（通常表示时间），若对每个 $t \in T$，定义在该概率空间上的随机变量 $X(t, \omega)$ 构成一族随机变量，则称 $\{X(t), t \in T\}$ 为随机过程。其中，$\omega \in \Omega$ 表示样本点。

核心数学表示：
- 对固定 $t$，$X(t)$ 是一个随机变量
- 对固定 $\omega$，$X(t)$ 是时间 $t$ 的一个样本函数（轨道）

离散时间随机过程可记为 $\{X_n, n = 0,1,2,\dots\}$，连续时间过程记为 $\{X(t), t \geq 0\}$。

## 二、核心原理

1. **状态空间与参数集**：随机过程由状态空间 $S$（随机变量取值的集合）和参数集 $T$（时间指标集）共同定义。状态空间可以是离散的（如整数集）或连续的（如实数集），参数集同样可以是离散或连续。

2. **马尔可夫性**：未来状态仅依赖于当前状态，与过去状态条件独立。数学表达为 $P(X_{n+1} = j | X_n = i, X_{n-1} = i_{n-1}, \dots, X_0 = i_0) = P(X_{n+1} = j | X_n = i)$。这是随机过程中最重要的性质之一，极大简化了分析复杂度。

3. **平稳性**：过程的统计特性不随时间平移而改变。严平稳要求任意有限维分布族在时间平移下不变；宽平稳仅要求均值函数为常数，协方差函数仅依赖于时间差。

4. **遍历性**：时间平均等于系综平均（期望）。即 $\lim_{T \to \infty} \frac{1}{T} \int_0^T X(t) dt = E[X(t)]$（几乎必然成立）。这是从单条样本轨道推断整体统计特性的理论基础。

5. **鞅性质**：给定当前信息，未来期望等于当前值：$E[X_{n+1} | X_0, \dots, X_n] = X_n$。鞅是公平博弈的数学模型，在金融数学中具有核心地位。

## 三、过程/示例

考虑一个简单的随机游走过程（离散时间、离散状态空间）：

**设定**：初始位置 $X_0 = 0$，每一步以概率 $p=0.5$ 向右移动+1，以概率 $q=0.5$ 向左移动-1。即 $X_{n+1} = X_n + \xi_n$，其中 $\xi_n = \pm 1$ 等概率。

**示例执行**（10步模拟）：

| 步数 n | 随机数 | 移动方向 | 位置 $X_n$ |
|--------|--------|----------|------------|
| 0      | -      | -        | 0          |
| 1      | 0.73   | +1       | 1          |
| 2      | 0.21   | -1       | 0          |
| 3      | 0.88   | +1       | 1          |
| 4      | 0.45   | -1       | 0          |
| 5      | 0.92   | +1       | 1          |
| 6      | 0.11   | -1       | 0          |
| 7      | 0.67   | +1       | 1          |
| 8      | 0.34   | -1       | 0          |
| 9      | 0.82   | +1       | 1          |
| 10     | 0.56   | +1       | 2          |

**关键观察**：
- 该过程是马尔可夫链：$X_{n+1}$ 仅依赖于 $X_n$
- 均值 $E[X_n] = 0$（对称随机游走）
- 方差 $Var[X_n] = n$（随时间线性增长）
- 该过程是鞅：$E[X_{n+1} | X_n] = X_n$

## 四、复杂度/性质分析

1. **时间复杂度**：模拟 $N$ 步随机过程的时间复杂度为 $O(N)$，每步仅需常数次随机数生成和状态更新。对于马尔可夫链蒙特卡洛（MCMC）方法，收敛所需步数通常为 $O(\log(1/\epsilon))$ 到 $O(1/\epsilon^2)$ 量级。

2. **空间复杂度**：存储完整样本轨道需 $O(N)$ 空间；若仅需当前状态和统计量，可优化至 $O(1)$。

3. **收敛性**：
   - 马尔可夫链在不可约、非周期、正常返条件下收敛到平稳分布
   - 收敛速度由第二特征值 $\lambda_2$ 决定，混合时间 $\tau \approx 1/(1-|\lambda_2|)$
   - 大数定律保证样本均值收敛到期望值（遍历性）

4. **稳定性**：对于线性随机微分方程 $dX_t = aX_t dt + \sigma dW_t$，当 $a < 0$ 时过程均值回归稳定；$a > 0$ 时指数发散。

5. **误差分析**：蒙特卡洛模拟的均方误差为 $O(1/\sqrt{N})$，与维度无关（维数灾难的免疫性）。

## 五、常见误区

1. **混淆随机过程与时间序列**：随机过程是概率论框架下的理论模型，强调概率空间和随机变量族；时间序列是统计学的数据分析方法，侧重从观测数据中建模和预测。随机过程是时间序列的理论基础，但时间序列分析通常假设过程是平稳的。

2. **误认为马尔可夫性意味着无记忆**：马尔可夫性仅指给定现在，未来与过去独立，但过程本身可以具有长期记忆效应（如通过状态空间编码历史信息）。高阶马尔可夫链可通过扩展状态空间转化为一阶马尔可夫链。

3. **混淆严平稳与宽平稳**：严平稳要求所有有限维分布不变，条件极强；宽平稳仅要求一阶矩和二阶矩平稳。高斯过程中两者等价，但一般情况严平稳蕴含宽平稳，反之不成立。

4. **忽视遍历性条件**：并非所有平稳过程都是遍历的。例如，常数随机过程 $X(t) = Z$（$Z$ 为随机变量）是平稳的，但时间平均等于 $Z$（随机变量），不等于系综平均 $E[Z]$。遍历性需要额外的混合条件。

5. **错误理解鞅与马尔可夫链的关系**：鞅是条件期望性质（无漂移），马尔可夫链是条件独立性性质。两者正交：存在非马尔可夫的鞅（如累积和过程），也存在非鞅的马尔可夫链（如带漂移的随机游走）。
