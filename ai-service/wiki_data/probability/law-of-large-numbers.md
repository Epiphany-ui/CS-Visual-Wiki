# 大数定律

## 一、定义

大数定律（Law of Large Numbers, LLN）是概率论中描述随机试验结果平均值的收敛性的基本定理。它指出：当独立同分布的随机变量序列的样本量趋于无穷大时，样本均值依概率收敛于其数学期望（总体均值）。

**核心数学公式**：

设 \(X_1, X_2, \ldots, X_n\) 为独立同分布的随机变量序列，且 \(E[X_i] = \mu\) 存在有限。定义样本均值 \(\bar{X}_n = \frac{1}{n}\sum_{i=1}^n X_i\)，则：

**弱大数定律**（辛钦定理）：
\[
\lim_{n \to \infty} P(|\bar{X}_n - \mu| > \varepsilon) = 0, \quad \forall \varepsilon > 0
\]

**强大数定律**：
\[
P\left(\lim_{n \to \infty} \bar{X}_n = \mu\right) = 1
\]

## 二、核心原理

1. **频率与概率的桥梁**：大数定律为统计学中“用样本估计总体”提供了理论基础。它解释了为什么抛硬币次数越多，正面频率越接近1/2。

2. **依概率收敛 vs 几乎必然收敛**：弱大数定律保证当n足够大时，样本均值偏离期望的概率可以任意小；强大数定律则保证样本均值几乎必然收敛到期望，两者在收敛强度上有本质区别。

3. **独立同分布假设**：核心前提是随机变量相互独立且服从相同分布。若违反独立性（如时间序列数据）或同分布性（如混合分布），大数定律可能不成立。

4. **方差条件**：弱大数定律仅要求期望存在（辛钦定理），而切比雪夫大数定律要求方差存在且有界。强大数定律通常要求方差有限（柯尔莫哥洛夫定理）。

5. **收敛速度**：大数定律不给出具体收敛速率，但中心极限定理补充了这一点——样本均值的分布近似正态分布，标准差以 \(1/\sqrt{n}\) 衰减。

## 三、过程/示例

**场景**：模拟掷一个公平六面骰子，期望值 \(\mu = 3.5\)。

**步骤1**：生成随机序列
掷10次骰子，记录结果：[4, 2, 6, 1, 3, 5, 2, 4, 6, 1]

**步骤2**：计算累积均值
- n=1: \(\bar{X}_1 = 4/1 = 4.0\)
- n=2: \(\bar{X}_2 = (4+2)/2 = 3.0\)
- n=3: \(\bar{X}_3 = (4+2+6)/3 = 4.0\)
- n=4: \(\bar{X}_4 = (4+2+6+1)/4 = 3.25\)
- n=5: \(\bar{X}_5 = (4+2+6+1+3)/5 = 3.2\)
- n=6: \(\bar{X}_6 = (4+2+6+1+3+5)/6 = 3.5\)
- n=7: \(\bar{X}_7 = (4+2+6+1+3+5+2)/7 \approx 3.29\)
- n=8: \(\bar{X}_8 = (4+2+6+1+3+5+2+4)/8 = 3.375\)
- n=9: \(\bar{X}_9 = (4+2+6+1+3+5+2+4+6)/9 \approx 3.67\)
- n=10: \(\bar{X}_{10} = (4+2+6+1+3+5+2+4+6+1)/10 = 3.4\)

**步骤3**：观察收敛
随着n从1增加到10，样本均值从4.0波动到3.4，逐渐接近理论值3.5。当n=1000时，均值通常在3.45-3.55之间；n=10000时，均值几乎稳定在3.5附近。

## 四、复杂度/性质分析

1. **收敛速度**：大数定律不保证具体收敛速率，但中心极限定理指出收敛速度为 \(O(1/\sqrt{n})\)。即误差以样本量平方根的倒数衰减。

2. **样本量需求**：要达到误差 \(\varepsilon\) 的置信水平 \(1-\alpha\)，所需样本量约为 \(n \approx (z_{\alpha/2} \cdot \sigma / \varepsilon)^2\)，其中 \(\sigma\) 为标准差。

3. **空间复杂度**：计算累积均值仅需 \(O(1)\) 额外空间（维护当前和与计数），无需存储全部样本。

4. **时间复杂度**：在线计算均值为 \(O(n)\)，每次更新为 \(O(1)\)。

5. **鲁棒性**：对异常值敏感——单个极端值会显著影响小样本均值，但随着n增大，影响逐渐稀释。

6. **收敛模式**：弱收敛（依概率）比强收敛（几乎必然）条件更弱，但实际应用中通常满足强收敛条件。

## 五、常见误区

1. **“大数定律保证小样本也接近期望”**：错误。大数定律是渐近性质，小样本时均值可能严重偏离期望。例如掷3次骰子全得6，均值为6，远偏离3.5。

2. **“大数定律与中心极限定理等价”**：混淆。大数定律描述均值收敛到期望，中心极限定理描述均值分布的形状（正态分布）。前者是点估计，后者是分布估计。

3. **“赌徒谬误”**：认为独立事件有“补偿”机制。例如连续5次硬币正面后，认为下一次更可能反面。大数定律只保证长期频率，不保证短期补偿。

4. **“样本量足够大时，所有随机现象都消失”**：错误。大数定律只保证均值收敛，但单个随机变量的波动性依然存在。方差大的分布需要更大样本才能稳定。

5. **“大数定律适用于任何随机变量序列”**：错误。需要独立同分布或至少满足特定条件（如马尔可夫链的遍历性）。相关序列（如股票价格）可能不满足经典大数定律。
