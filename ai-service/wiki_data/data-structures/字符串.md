# 字符串

## 一、定义
字符串（String）是定义在有限字母表 Σ 上的有限序列。形式上，字符串 s 可表示为 s = a₁a₂…aₙ，其中每个 aᵢ ∈ Σ，n = |s| 称为字符串的长度。空字符串 ε 是长度为 0 的特殊字符串。在计算机科学中，字符串通常以字符数组或链表形式存储，其核心操作包括连接、子串提取、模式匹配等。

## 二、核心原理
1. **字符编码与存储**：字符串的底层实现依赖于字符编码系统（如ASCII、Unicode）。每个字符对应一个整数编码，字符串在内存中通常以连续地址存储（C风格以'\0'结尾），或采用长度前缀方式（Pascal风格）。现代语言（如Java、Python）采用对象封装，支持不可变或可变两种语义。

2. **子串与模式匹配**：子串是原字符串中连续的一段。模式匹配问题（如KMP算法）的核心是利用已匹配部分的信息避免回溯。关键性质包括：前缀函数（π函数）记录模式串中每个位置的最长相等真前缀与真后缀长度，从而在失配时高效跳转。

3. **字符串哈希**：通过将字符串映射为整数（如多项式哈希：H(s) = Σ aᵢ × pⁱ mod M），可在O(1)时间内比较子串是否相等。但需处理哈希冲突，通常采用双哈希或滚动哈希（Rabin-Karp算法）。

4. **字典序与排序**：字符串的字典序基于字符编码顺序逐位比较。后缀数组（Suffix Array）和字典树（Trie）是处理字符串排序与检索的核心数据结构，可在O(n log n)时间内完成排序。

5. **不可变性与驻留**：许多语言（如Java、Python）的字符串是不可变的，这保证了线程安全与哈希缓存。字符串驻留（String Interning）机制使得相同字面量共享同一内存，优化存储效率。

## 三、过程/示例
以KMP算法在文本串T="ABABABC"中查找模式串P="ABABC"为例：

**步骤1：计算前缀函数π**  
模式串P="ABABC"的π数组：  
- π[0]=0（A无真前缀）  
- π[1]=0（AB：前缀"A"≠后缀"B"）  
- π[2]=1（ABA：前缀"A"=后缀"A"）  
- π[3]=2（ABAB：前缀"AB"=后缀"AB"）  
- π[4]=0（ABABC：无相等前后缀）  
得到π=[0,0,1,2,0]

**步骤2：匹配过程**  
初始化i=0（文本指针），j=0（模式指针）  
1. T[0]=A，P[0]=A → 匹配，i=1，j=1  
2. T[1]=B，P[1]=B → 匹配，i=2，j=2  
3. T[2]=A，P[2]=A → 匹配，i=3，j=3  
4. T[3]=B，P[3]=B → 匹配，i=4，j=4  
5. T[4]=A，P[4]=C → 失配，j=π[3]=2（不回溯i）  
6. T[4]=A，P[2]=A → 匹配，i=5，j=3  
7. T[5]=B，P[3]=B → 匹配，i=6，j=4  
8. T[6]=C，P[4]=C → 匹配，j=5=len(P) → 匹配成功，位置i-j+1=2

## 四、复杂度/性质分析
1. **时间复杂度**：  
   - 朴素匹配：O(n×m)，最坏情况如T="AAAAAB"，P="AAAB"  
   - KMP算法：O(n+m)，线性时间，无论最好/最坏/平均  
   - 字符串哈希匹配：平均O(n+m)，最坏O(n×m)（哈希冲突时）  
   - 后缀数组构建：O(n log n)（倍增法）或O(n)（DC3算法）

2. **空间复杂度**：  
   - KMP算法：O(m)用于存储π数组  
   - 后缀数组：O(n)用于存储排序结果  
   - 字典树：O(Σ × 节点数)，最坏O(n×Σ)

3. **稳定性**：字符串排序算法（如基数排序）是稳定的，但基于比较的排序（如快排）不稳定。

4. **收敛性**：字符串匹配算法保证有限步内终止，因为每次比较要么推进指针，要么通过π函数跳转，不会陷入无限循环。

## 五、常见误区
1. **空字符串与null混淆**：空字符串""是长度为0的有效对象，而null表示不存在引用。在Java中，调用null字符串的length()方法会抛出NullPointerException。

2. **字符串不可变性的误解**：虽然字符串对象不可变，但变量可以重新赋值。例如Python中s="hello"; s+=" world"实际创建了新对象，原对象仍存在。

3. **字符编码的陷阱**：Unicode字符可能占用多个代码单元（如UTF-16中的代理对），导致len()函数返回的是代码单元数而非实际字符数。例如emoji"😊"在Java中长度为2。

4. **模式匹配的回溯误区**：初学者常认为KMP算法完全不回溯，实际上文本指针从不后退，但模式指针通过π函数跳转，本质是“利用已匹配信息避免无效回溯”。

5. **字符串比较的引用与值**：在Java中，==比较的是引用地址，equals()比较内容。String s1="abc"; String s2=new String("abc"); s1==s2为false，但s1.equals(s2)为true。Python中==比较内容，is比较引用。
